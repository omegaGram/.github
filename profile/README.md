# omegaGram

> Telegram-compatible server implementation written in Rust.

![Status](https://img.shields.io/badge/status-in%20development-blue)
![Language](https://img.shields.io/badge/language-Rust-orange)
![Protocol](https://img.shields.io/badge/protocol-MTProto-green)
![Platform](https://img.shields.io/badge/clients-Android%20%7C%20Desktop-purple)

`omegaGram` is a self-written Telegram-compatible backend built around the MTProto protocol. The project implements the server side required for official Telegram clients to work on independent infrastructure while remaining compatible with the original applications.

## current status

The project already has a solid foundation and implements most of the core functionality.

### messaging

- almost complete private messaging
- groups (almost complete)
- media sending
- reactions
- stickers
- emoji packs
- emoji statuses

### account

- fully implemented profile editing
  - profile photos
  - first & last name
  - username
  - bio
  - and more
- basic privacy settings

### clients

- Android client support
- Telegram Desktop support

### other

- many smaller Telegram features have already been implemented

## roadmap

The following features are planned before the first stable release:

- complete private messaging implementation
- complete groups implementation
- complete channels implementation
- full **Star Gifts** support
  - purchasing
  - upgrading
  - transferring
  - auctions
  - and more
- Bot API
- BotFather
- voice calls
- channel livestreams
- supergroups
- remaining Telegram functionality
